Experiments to Launch Cash App on Page Load

Page Description Behavior
cash-app-button.html Button rendered and totally managed by Cash API Button When clicked on phone, directly launches Cash App
custom-app-button.html Custom button, rendered and managed by us When clicked on phone, directly launches Cash App
window-load.html Attempt to launch app on page load When opened on phone, on page load automatically loads Cash App interstitial page
universal-link.html Directly use Universal Link without Cash Pay API involved. Demonstrates that not being able to launch app on page load is due to an iOS-level restriction When opened on phone, in 10 seconds automatically loads Cash App interstitial page, but custom button launches Cash App.
